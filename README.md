## Доступные команды (scripts)
Установить необходимые зависимости, если у вас их нет:
#### `npm install`
Запустить json-server::
#### `npm start`
developer сборка:
#### `npm run dev`
production сборка: 
#### `npm run build`
Развернуть проект на github pages: 
#### `npm run deploy`

## Запуск:
    npm install
    npm start
